12 
  
Le tableau périodique
1.
 
Comment les éléments sont-ils classés dans la classifi-
cation périodique ?
Ils sont rangés par numéro atomique croissant.
2.
 
Comment appelle-t-on les lignes ?
On les appelle des périodes.
3.
 
Combien y a-t-il de lignes ? 
Il y en a sept.
4.
 
Comment appelle-t-on les colonnes ?
Les éléments situés dans une même colonne consti-
tuent une famille chimique.
5.
 
Quelle est la relation entre les éléments d’une même 
famille ? 
Ils ont des propriétés chimiques analogues.
6.
 
Quelle est la relation entre les structures électroniques 
des éléments d’une même famille ?
Ils ont le même nombre d’électrons sur leur couche 
électronique externe (périphérique).
